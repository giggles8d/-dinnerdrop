## Video Production Notes

### Option A — Voiceover only (fastest to launch)
- Record VO using ElevenLabs (Rachel voice — warm, female, relatable)
- Combine with stock footage: tired parent at fridge (Pexels), app screen recordings (record locally)
- Edit in CapCut or DaVinci Resolve free
- Total production time: ~2 hours once script approved

### Option B — Screen recording demo (highest converting)
- Record DinnerDrop app screen on mobile or desktop browser
- Add VO in post, text overlays on key moments
- Most effective format for SaaS/app YouTube ads
- Requires: working DinnerDrop account, screen recording software

### Minimum viable creative (unblocked today):
Use Script V2 screen demo format — no talent, no stock footage, just a clean screen recording + Rachel VO from ElevenLabs. This is the fastest path to a live YouTube ad.
