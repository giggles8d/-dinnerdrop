## Overview

Google Display Network reaches 90%+ of internet users across 2M+ websites, apps, and Google properties. Responsive Display Ads (RDAs) let Google automatically mix and match your headlines, descriptions, images, and logos to find the highest-performing combinations.

**DinnerDrop Display Strategy:**
- Run Display as a retargeting layer only in Month 1 (target: users who visited /beta but did not sign up)
- Budget: $0/day Week 1–2 (run Search first, build pixel data), $2–3/day Week 3+ for retargeting
- Primary goal: bring back warm traffic cheaply (display CPCs typically $0.20–$0.80 vs $1.50–$2.50 Search)
- Do NOT run cold-audience display in Month 1 — low conversion rates without social proof/reviews

## Retargeting Audience Setup

Before Display ads go live, Sarah must create these audiences in Google Ads:

### Audience 1 — /beta Page Visitors (did not convert)
- Source: Google Ads tag (website visitors)
- URL contains: `/beta`
- Exclude: users who completed checkout (URL contains `/beta/success` or event: beta_signup)
- Membership duration: 30 days
- **Use for:** Display retargeting, bid adjustment +20%

### Audience 2 — All Site Visitors
- Source: Google Ads tag (all pages)
- Membership duration: 14 days
- **Use for:** Broad retargeting, lower bids, reach extension

### Audience 3 — Similar Audiences (auto-generated by Google)
- Google auto-creates these once Audience 1 reaches 1,000+ members
- **Use for:** Prospecting in Month 2

## Display Campaign Structure

### Campaign: DinnerDrop Display — Retargeting
- **Type:** Display
- **Goal:** Website traffic / Conversions
- **Bid strategy:** Target CPA $12 (once enough conversion data exists) / Manual CPC at launch
- **Start date:** Week 3 (after Search campaign has run 2 weeks and built pixel data)
- **Daily budget:** $2–3/day (scale to $5 at 50+ conversions)

### Ad Group 1 — /beta Page Abandoners
- **Audience:** Audience 1 (/beta visitors, did not convert)
- **Ads:** RDA using all assets above
- **Bid:** $0.50 max CPC manual (retargeting = warm audience, low CPCs expected)
- **Frequency cap:** 5 impressions/user/day

### Ad Group 2 — All Site Visitors (Week 4+)
- **Audience:** Audience 2 (all site visitors)
- **Ads:** RDA using landscape + square images, long headlines, descriptions
- **Bid:** $0.30 max CPC
- **Frequency cap:** 3 impressions/user/day
